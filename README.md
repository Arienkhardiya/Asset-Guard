# 🚀 Asset Guard AI

**AI-Powered Anti-Piracy & Digital Asset Protection Platform**

---

## 📌 Overview

Asset Guard AI is a full-stack web platform designed to help creators, organizations, and enterprises **detect, monitor, and prevent unauthorized distribution of digital content**.

The system uses **AI-driven insights, secure authentication, and cloud infrastructure** to provide real-time tracking and protection of assets.

---

## 🎯 Problem Statement

Digital content piracy leads to massive revenue loss and intellectual property violations.
Existing solutions are either:

* Manual
* Slow
* Not scalable

---

## 💡 Our Solution

Asset Guard AI automates the entire process using:

* AI-based detection
* Real-time monitoring
* Secure workspace-based access
* Centralized dashboards

---

## ⚡ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/assetguard.git
cd assetguard
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file:

```env
PORT=8080
JWT_SECRET=your_secret
DATABASE_URL=your_postgres_url
GEMINI_API_KEY=your_gemini_key
```

### 4️⃣ Run Backend

```bash
npm start
```

### 5️⃣ Run Frontend

```bash
npm run dev
```
